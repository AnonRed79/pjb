# Python Java Builder

This is a project I'm build because I don't like gradle. It's strange and unintuitive, so I'm going in the oposite direction with this. It's built with python, because it's a simple and well known language so it should be easy to use, it can easily work with dedicated pjb repositories, or it can simple download a specified dependancie. I'm also planing to make plug-ins for common java IDEs such as VS Code and IntelliJ, but that's for later, for now it's just a cli tool.

Also of note, I'm going to mainly develop this for linux, with windows being secondary, and I don't have any way to test on Mac as of now. I may look into it in the future.
